The interpreter used to interpret statements used in the programming interface is based 
on the basic interpreter provided by `Jay Conrod` in his great tutorial.

The Tutorial, original code and further information can be found on 
[his website](https://jayconrod.com/posts/37/a-simple-interpreter-from-scratch-in-python--part-1-)
